# 2. Teilnahme an planbaren Einsätzen bei Feuerwehren (speziell im Bereich Stabsarbeit)

- According to the author's experience, the number of scheduled fire brigade deployments in most fire brigades in Germany is so low that a percentage of the JAZ of fire brigade instructors in the order of 8.46% of the JAZ does not seem to be achievable by participating in these deployments. Furthermore, it is not always possible for such locations to occupy interesting operational functions useful for the respective fire brigade instructors.

- Furthermore, it is not always possible for such positions to occupy interesting functions for the respective fire fighting instructors usefully. Example: at a large event, which is processed within the framework of the leadership stage D, the tasks of a train leader, which is used within the framework of this large event, can differ significantly from the usual and interesting tasks for the fire fighting instructor.

- For example, in a major event, which is carried out within the framework of the leadership stage D, the tasks of a train driver, which is used within the framework of this major event, can differ considerably from the usual tasks which are interesting for the fire brigade instructor. For these reasons, this possibility was not chosen.
