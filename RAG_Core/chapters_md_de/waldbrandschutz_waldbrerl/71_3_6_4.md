# 3.6.4

- Die LFoA übergibt die jeweils aktualisierten Daten kostenfrei an das LPBK zur Bereitstellung für die  Landkreise, kreisfreien Städte und Gemeinden. Weiterhin übergibt die LFoA diese Daten dem LAiV für  die Führung im Geoportal des Landes.

- Weiterhin übergibt die LFoA diese Daten dem LAiV für  die Führung im Geoportal des Landes. Damit wird ein öffentlicher Zugang auf die digitale Ausgabe der  Waldbrandeinsatzkarte ermöglicht.
