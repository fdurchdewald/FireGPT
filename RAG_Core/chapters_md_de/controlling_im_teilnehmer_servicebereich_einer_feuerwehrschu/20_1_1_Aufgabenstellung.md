# 1.1 Aufgabenstellung

- Controlling im Teilnehmer-Servicebereich einer Feuerwehrschule –   Entwickeln Sie ein Berichtswesen und Kennzahlensystem, das im Rahmen des  Controllings einer staatlichen Feuerwehrschule im Bereich des Teilnehmer-  Service (Platzvergabe, Unterbringung, Versorgung) eingesetzt werden kann.
