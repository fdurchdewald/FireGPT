# 3.2.3 Berichtswesen

- Ein Berichtswesen hat, in diesem Fall bezogen auf den Teilnehmer-Service, die  Aufgabe der Bündelung und Bereitstellung von Informationen, die steuerungsre- levant für den Betrieb einer Feuerwehrschule sind.
