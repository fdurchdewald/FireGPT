# 2. Einsatz-

- vorbereitung  18 19 Die 2-5er Regel beschreibt eine Faustregel zur Führbarkeit von taktischen Einheiten. Kernaus­ sage der Regel ist, dass eine Führungskraft zwischen zwei und fünf taktische Einheiten der  untergeordneten Führungsebene führen kann.

Mehr als fünf taktische Einheiten müssen auf  mehrere Führer/Unterführer verteilt werden, welche dann von der übergeordneten Führungs­ kraft geführt werden.
