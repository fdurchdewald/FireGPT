# 4.	 Gefährdung/Schutz vollbrandgefährdeter oder schnell verbrennender Vegetation

- Diese Priorisierung bildet die Grundlage für die weitere taktische Vorgehensweise. In den meis­ ten Fällen eines Vegetationsbrandes besteht keine Gefahr für Betroffene oder Verletzte, so dass  diese Fälle keine zeitkritischen Ereignisse darstellen, um vordergründig eine Menschenrettung  einzuleiten.
